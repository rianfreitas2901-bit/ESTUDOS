CSS External

Um CSS externo, como o própio nome sugere, está fora do documento HTML.

As principais vantagem no uso desse metódo são:

  • Como o arquivo está "externo", ou seja, fora do arquivo .html, o seu conteúdo fica separado do estilo, e isso deixa tudo mais organizado.
  • Ainda podemos permanecer aplicando os diversos estilo em cascata. Ainda sendo possível aolicar as duas outras estruturas.
  • Um mesmo arquivo de estilo pode ser utilizado em várias páginas do seu sistema/site. Vamos aplicar também os estilos na home do nosso exercício.
